About the Data

🎵 Spotify Audio Features

Every track is analyzed using Spotify's audio feature API — energy, valence, acousticness, instrumentalness, danceability, speechiness, tempo, key, and mode. We pull the top 100 tracks per genre to build statistically meaningful profiles.

📊 Statistical Analysis

Raw features are processed into distributions, scatter plots, correlations, and comparative breakdowns. Median values over means to reduce outlier influence. Every chart tells a story about what makes a genre unique.

🤖 AI-Powered Insights

Gemini analyzes patterns humans might miss — genre boundaries, emotional fingerprints, production trends, and artist clustering. These insights feed back into kapiko's own music generation pipeline.
